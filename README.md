# 🚗 AutoParser — Парсер автомобильных площадок и аукционов

Модульная система для автоматического сбора объявлений о продаже автомобилей и мототехники с международных площадок. Приложение работает автономно, имеет современный графический интерфейс (GUI) и поддерживает экспорт в Excel, CSV и JSON.

---

## 🌟 Особенности

- 🖥️ **Графический интерфейс (GUI)** — удобный тёмный интерфейс на CustomTkinter со встроенной таблицей результатов и открытием ссылок в 1 клик.
- ⚡ **Автономность** — работа без сторонних баз данных: скачивает, фильтрует и выводит результат прямо в приложение.
- 🛡️ **Обход антибот-систем** — поддержка Playwright Stealth, маскировка `navigator.webdriver`, ротация User-Agent и задержек.
- 🇦🇪 **DubiCars (ОАЭ)** — точный парсинг цен (AED), спецификаций, пробега, типов топлива и локаций.
- 📊 **Гибкий экспорт** — сохранение результатов в файлы Excel (`.xlsx`), CSV (`.csv`) и JSON (`.json`).
- ⚡ **CLI режим** — поддержка запуска из командной строки для скриптов и автоматизации.

## 🚀 Быстрый старт

### 1. Требования
- macOS / Windows / Linux
- Python 3.10 или новее

### 2. Запуск графического интерфейса (GUI)

Откройте Терминал в папке проекта и выполните:

```bash
cd /Users/zakhar/Desktop/AutoParser
./venv/bin/python gui.py
```

В открывшемся окне:
1. Введите поисковый запрос (например: `Mitsubishi Outlander`).
2. Укажите интервал лет (например: от `2022` до `2026`).
3. Укажите количество страниц (например: `3`).
4. Нажмите **🚀 Начать поиск**.

## 💻 Использование через командную строку (CLI)

```bash
# Поиск Mitsubishi Outlander (2022-2026 гг.) на DubiCars (ОАЭ)
./venv/bin/python main.py -p dubicars -q 'Mitsubishi Outlander' --min-year 2022 --max-year 2026 --pages 3 -o excel

# Поиск Toyota Land Cruiser в фоновом режиме (без открытия браузера)
./venv/bin/python main.py -p dubicars -q 'Toyota Land Cruiser' --pages 5 --headless

# Поиск по всем площадкам сразу
./venv/bin/python main.py -p all -q 'BMW X5' -o json
```

### Параметры командной строки

| Аргумент | Описание | Пример |
|---|---|---|
| `-p`, `--platform` | Площадка (`dubicars`, `autoscout24`, `bidcars`, `encar`, `che168`, `autohome`, `all`) | `-p dubicars` |
| `-q`, `--query` | Поисковый запрос (марка / модель) | `-q 'Toyota Camry'` |
| `--min-year` | Минимальный год выпуска | `--min-year 2022` |
| `--max-year` | Максимальный год выпуска | `--max-year 2026` |
| `--pages` | Максимальное количество страниц | `--pages 3` |
| `-o`, `--output` | Формат сохранения (`excel`, `csv`, `json`) | `-o excel` |
| `--headless` | Запуск браузера в фоновом режиме | `--headless` |
| `--proxy-file` | Путь к файлу со списком прокси | `--proxy-file proxies.txt` |

## 📋 Модель данных (VehicleListing)

Все объявления приводятся к единому формату из 19 полей:

| Поле | Тип | Описание |
|---|---|---|
| `source` | str | Название площадки (`dubicars`, `autoscout24`, ...) |
| `listing_id` | str | Уникальный ID объявления |
| `url` | str | Прямая ссылка на карточку авто |
| `title` | str | Полное название объявления |
| `make` | str | Марка автомобиля (Toyota, Mitsubishi...) |
| `model` | str | Модель (Outlander, Land Cruiser...) |
| `trim` | str | Комплектация |
| `year` | int | Год выпуска |
| `price` | float | Цена |
| `currency` | str | Валюта (AED, EUR, USD, KRW, CNY) |
| `mileage_km` | int | Пробег в километрах |
| `fuel_type` | str | Тип топлива (Petrol, Hybrid, Electric) |
| `transmission` | str | Коробка передач (Automatic, Manual) |
| `body_type` | str | Тип кузова (SUV, Sedan...) |
| `color` | str | Цвет кузова |
| `location` | str | Местоположение / Город |
| `seller_type` | str | Тип продавца (Dealer / Private) |
| `image_url` | str | Ссылка на главное фото |
| `parsed_at` | datetime | Дата и время сборки данных |

## 🔒 Прокси и Защита от блокировок

Для добавления прокси-серверов создайте файл `proxies.txt` в корне проекта:

```text
http://123.45.67.89:8080
http://user:password@98.76.54.32:3128
socks5://10.20.30.40:1080
```

> **Примечание по площадкам Китая (che168, autohome)**: Для работы с китайскими сайтами требуются резидентные прокси Китая, так как данные сервисы применяют гео-блокировку для не-китайских IP-адресов.
